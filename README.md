# leetcode-sql-solutions
This repository contains my SQL solutions for LeetCode database problems.
It covers core DBMS concepts such as SELECT queries, JOINs, GROUP BY,
aggregate functions, subqueries, and filtering.

## Purpose
- Strengthen SQL and database fundamentals
- Practice real-world query scenarios
- Prepare for technical interviews

## Topics Covered
- Basic SELECT & WHERE
- JOINS (INNER, LEFT, SELF JOIN)
- GROUP BY & HAVING
- Subqueries
- Data filtering and aggregation

## Platform
- LeetCode (Database / SQL problems)
